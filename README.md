## Sobre mí 👨‍💻

Soy desarrollador fullstack en formación, con experiencia construyendo aplicaciones web. Me apasiona trabajar en proyectos concretos que resuelvan problemas reales — desde APIs REST hasta PWAs con sincronización en tiempo real.

Me muevo cómodo entre el backend y el frontend, y disfruto especialmente resolver problemas de integración complejos: conexiones a múltiples bases de datos, sistemas de autenticación y permisos granulares, y optimización de flujos de trabajo internos.

Siempre aprendiendo. Siempre construyendo.
